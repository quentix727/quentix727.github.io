Minecraft Java Edition 1.21.1安装教程

预备工作（可选）：

检查系统配置。同时按住Windows()+R键，输入“cmd”（图1），在弹出的命令提示符窗口中输入“systeminfo>%userprofile%\Desktop\info.txt”并按回车，稍等片刻后桌面上会出现一个名为“info.txt”的文本文档（图2），你可以参考Minecraft官网的要求，也可以询问我。

点击这里下载安装Minecraft Java Edition 1.21.1（以下简称MC）所必须的压缩包Minecraft.zip（密码：4uw9ew）。

                  

图1                                                                图2

1.       安装Java

双击压缩包中的“jdk-22_windows-x64_bin.exe”，根据提示一路点击“下一步”，安装完成后就可以点击“完成”按钮。

2.       配置PCL启动器

创建一个启动器专用文件夹（路径中尽量避免含有中文和特殊字符），复制“Plain Craft Launcher 2.exe”并启动后，点击“同意”按钮同意协议。切换到“下载”标签页，点击“1.21.1”版本，点击“开始安装”按钮（图3），安装完成后返回主页，在唯一的输入框里起一个心仪的名字，点击启动游戏。

图3

3.       加入服务器

打开游戏（图4）后点击多人游戏，点击“同意”，点击“直接连接”（图5），输入我给你的服务器地址，点击“加入服务器”，你就可以畅快地玩耍了！
